📌 Overview

This is a simple console-based Quiz Application built using Java.
The application presents multiple-choice questions to the user, accepts input, evaluates answers, and displays the final score.

It also allows the user to replay the quiz.

🚀 Features

Multiple-choice questions

User input using Scanner

Score calculation

Result feedback based on performance

Shuffle questions each round

Option to replay the quiz

🛠 Technologies Used

Java

Console (Command Line Interface)

Java Collections Framework (List, ArrayList, Collections.shuffle())
